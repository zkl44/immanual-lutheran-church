Directions

  1. Make sure that file naming follows correct format. ( saved as a PDF, not a DOC )
    • If new Bulletin or Announcement - save as day of month in 2 digit format ( 01.pdf, 18.pdf, 31.pdf ... )
    • If new Calendar - save as Month ( June.pdf )
  2. Select the intended options for items 1 - 3.
  3. Choose the PDF from your machine that you wish to upload.
  4. Upload, you're done!
